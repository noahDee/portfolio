Personal Pokedex

One of my first projects, inspired by pokemon, this is a Pokedex built using HTML, CSS, and JavaScript. It allows the user to both view an entered pokemon's stats and add them to their personal roster. These stats are from data extracted from a pokemon API and through an AJAX call, is then shown on screen through dynamic stats bars. I particulary enjoyed looking through and playing with the API and coming up with a way to populate that data onto the screen in a manner that fit the theme.

Junglr

A tumblr inspired, animal themed, social media platform where users can post and view articles. Articles can be marked with tags, making them easier to find based on your interests. This website was made using HTML, CSS, Ruby and Sinatra. This was my intro to building a full-stack webpage and enjoyed learning about databases, routing, authentication and more. I also enjoyed implementing the tag system as a way for users to be able to keep track of their favorite topics and articles.

Community Chest

This project was a collaborative effort between four other team members and myself. This website allows users to post about items they are willing to give away for free. The amount of posts a user can see is limited unless they purchase a premium account which can be done through the site. My personal contribution was the addition of the messaging system and setting up and troubleshooting the database tables and associations on the back end. I enjoyed working with my teammates to bring this idea to life. Created using HTML, CSS, JavaScript, Stripe, and Ruby on Rails.
